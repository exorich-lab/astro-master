# Astro + shadcn/ui Project

## Components
- Button: `src/components/ui/button.tsx`

## Add Components
```bash
npx shadcn@canary add component-name
```

## Tech Stack
- Astro v5.13.2

- Tailwind CSS v4.1.3

> For a complete list of technologies and versions, check the package.json file.
